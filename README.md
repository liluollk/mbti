# MBTI 人格测试

一个基于 React 的 MBTI 人格测试应用。

## 快速开始

### 环境要求
- Node.js >= 18.x
- npm >= 9.x

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 项目结构
```
src/
├── components/       # React 组件
├── data/            # 测试题目和描述数据
├── pages/           # 页面组件
├── utils/           # 工具函数
├── App.tsx          # 应用入口
└── main.tsx         # React 入口
```

## 功能特性
- 🎯 93道专业测试题目
- 📊 四维度性格分析
- 📝 详细的人格描述
- 💼 职业建议
- 💑 恋爱关系分析

## 技术栈
- React 18
- Vite 6
- Tailwind CSS 3
- React Router DOM 7
- TypeScript

## 注意事项
⚠️ 测试结果仅供娱乐参考，不构成专业心理评估。
